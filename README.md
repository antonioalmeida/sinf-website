## Getting Started

### Prerequisites
- Node.js (^8.16.0)
- npm (^6.10.1)

### Installation 

```bash
$ git clone https://github.com/semanadeinformatica/website
$ cd website

# install dependencies
$ npm install 

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```
